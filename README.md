# DroidCSS

LSPosed module for my modifications to Android 14

# Features

* Revert Material Components switches (e.g in Settings) to the system ones
* Revert Material Components buttons (e.g in some apps) to the system ones
* Replace the Quick Settings tiles with regular buttons

guaranteed to work. Support **might** be considered for later versions of Android with Material 3
